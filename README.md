## ⚡快速开始

您可以使用PyPI安装稳定版本，只需要在命令行输入以下命令:

```bash
$ pip install openvideo
```

或者通过Github获取最新版本:

```bash
$ pip install -U https://github.com/UmiMarch/OpenVideo/archive/master.zip # with --user for user install (no root)
```
